# 🌇 Dusk Dash

A fast-paced endless runner game built entirely with **HTML5 Canvas, CSS, and Vanilla JavaScript**. Run into the sunset, dodge obstacles, and survive as long as possible while increasing your score.

![Game Banner](https://img.shields.io/badge/Game-Endless%20Runner-orange)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?logo=javascript)

## 🎮 Features

- 🌅 Beautiful animated sunset environment
- 🏃 Smooth runner animation
- 🐢 Multiple obstacle types
  - Turtle
  - Dog
  - Dog Pack
  - Flying Goose
- ⚡ Speed Boost Mode
- 🛑 Brake Mode
- 📈 Progressive difficulty
- 🔊 Sound effects
- 📱 Mobile touch controls
- 💻 Responsive design
- 🏆 Best score tracking

---

## 🕹️ Controls

| Action | Keyboard |
|---------|----------|
| Jump | ↑ / W / Space |
| Duck | ↓ / S |

| Boost | → / D |

## 🎯 Gameplay

- Jump over ground obstacles.
- Duck under flying geese.
- Use **Boost** to earn points faster.
- Use **Brake** to survive difficult sections.
- Beat your highest score!
